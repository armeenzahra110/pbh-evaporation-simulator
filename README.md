 Theoretical Approximations & Future Extensions

The current iteration of this laboratory uses the standard semi-classical, semi-analytic approximation for Hawking dissipation, assuming emission dominated by massless fields (photons):

t_evap = (5120 * pi * G^2 * M^3) / (hbar * c^4)

 
Academic Caveat: High-Energy Particle Degrees of Freedom
In a complete astrophysical model (cf. Page 1976, MacGibbon & Webber 1991), the mass-loss rate dynamically couples to the thermodynamic Standard Model degrees of freedom. As the black hole's Hawking temperature surpasses the rest-mass energies of fundamental massive particles, emission channels open for neutrinos, leptons, and hadrons. 

This graybody tracking causes a real Primordial Black Hole to evaporate significantly faster in its terminal stages than this simplified photon-equivalent system predicts. This model serves as an undergraduate-level baseline approximation, establishing a clean, citable boundary for primordial dark matter constraints.
